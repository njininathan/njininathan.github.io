The technical projects below include personal, collaborative and research projects I have done. 

### Beta-proc
> This is an implementation of a multicycle CPU based on the Beta Instruction set from MIT.
> 
> It implements all 32 Instructions in the Instruction set using **Logisim**.
> 
> The CPU is improved to perform Digital Signal processing using a datapath similar to Texas Instrument's 1st DSP Processor. 
>>In later work, I intend to add Floating point number computations to the processor and redesign the processor using VHDL.
>
> Status: Completed in March 2021.
